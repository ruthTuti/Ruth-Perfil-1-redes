Respuestas a las Preguntas de Redes

Preguntas de la Actividad 1

1. ¿Qué son los componentes pasivos en la Red?

Los componentes pasivos en una red son aquellos que no requieren energía eléctrica para funcionar. Estos componentes se utilizan para transmitir señales sin modificarlas activamente.

2. Descripción técnica de cada componente pasivo con su respectiva imagen

  • RJ45: Conector estándar para cables Ethernet.
  • Jack o Keystone: Módulo de conexión para paneles de parcheo o placas de pared.
  • Cable UTP: Cable de par trenzado no blindado utilizado para transmisión de datos.
  • Patch Panel: Panel de conexiones para organizar y gestionar cables de red.
  • Patch cord: Cable de red corto para conexiones entre dispositivos y paneles.
  • Fibra óptica: Cable que utiliza luz para transmitir datos a alta velocidad.
  • Canaletas: Sistemas para organizar y proteger el cableado.

3. ¿Qué son los componentes activos en la Red?

Los componentes activos en una red son dispositivos electrónicos que requieren energía para funcionar y son capaces de procesar, amplificar o modificar las señales de red.

4. Descripción técnica de cada componente activo (Agregar imagen)

  • Switch: Dispositivo que conecta múltiples dispositivos en una red local y dirige el tráfico.
  • Router: Dispositivo que conecta diferentes redes y dirige el tráfico entre ellas.
  • Tarjeta de red: Componente que permite a un dispositivo conectarse a una red.
  • Interruptor inteligente Wifi: Switch con capacidades de gestión y configuración avanzadas, incluyendo funciones inalámbricas.
  • Access Point: Dispositivo que permite a dispositivos inalámbricos conectarse a una red cableada.

5. Herramientas utilizadas para realizar cableado estructurado

  • Ponchadora
  • Crimpadora
  • Peladora
  • Tester/probador de cables

6. ¿Qué es el cableado horizontal? Agregue imagen de referencia

El cableado horizontal es la porción del sistema de cableado que se extiende desde el área de trabajo hasta el cuarto de telecomunicaciones en el mismo piso.

7. ¿Qué es el cableado vertical o backbone?

El cableado vertical o backbone es la interconexión entre los cuartos de telecomunicaciones, cuartos de equipos e instalaciones de entrada en un edificio.

8. Explique los tipos de cables para el sistema de cableado horizontal

Los tipos principales son cable UTP (par trenzado no blindado) y fibra óptica. Cada uno tiene diferentes categorías y especificaciones según las necesidades de velocidad y distancia.

9. Investigue la normativa para hacer cables de red (clase A y B)

La normativa para cables de red incluye estándares como TIA/EIA-568, que define las especificaciones para cableado de telecomunicaciones en edificios comerciales.

10. ¿Para qué se utiliza un cable directo en la red?

Un cable directo se utiliza para conectar dispositivos de diferentes tipos, como un ordenador a un switch o un router a un módem.

11. ¿Para qué se utiliza un cable cruzado en la red?

Un cable cruzado se utiliza para conectar directamente dos dispositivos del mismo tipo, como dos ordenadores o dos switches.
